# register validator

```
npm run register-validator <operator-id-1> <operator-id-2> <operator-id-3> <operator-id-4> <keystore-file-name> <keystore-password>

```

example:
```
npm run register-validator 958 137 83 42 example-keystore.json 11111111
```
